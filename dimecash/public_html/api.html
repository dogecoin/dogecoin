This site has an unspendable api that can be called via a http get:

 https://dime.cash/cgi-bin/unspendable.cgi?first=D&name=DCxFiRSTxUNiTED&seed=30

or

> curl "https://dime.cash/cgi-bin/unspendable/cgi?first=D&name=DCxFiRSTxUNiTED&seed=30"
DCxDCxFiRSTxUNiTEDDxxxxxxxxxzCuKHf

The flags are:

first (D for Dogecoin)
name (Good luck, use little o, little i, capital L, and no 0) 
seed (30 for Dogecoin)
Right now, only dogecoin is supported, but you can download the python script and write your own wrapper or call it directly: https://github.com/johnrigler/unspendable

To learn more about the Base58Check format: https://en.wikipedia.org/wiki/Base58 Currently typechecking is not turned on, so if you enter an invalid character, this will throw up its hands. You can't use "I", "l", "0", or "1" because they are too similar looking to each other. The goal is the use as many capital letters as you can, using small letters "i" and "o". The small letter "x" is a space and "z" is a period.
