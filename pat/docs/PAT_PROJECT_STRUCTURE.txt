# =======================================================================
# DIRECTORY PURPOSES AND ORGANIZATION PRINCIPLES

# docs/ - Academic and publication materials
#   - Contains the complete academic paper with security proofs and benchmarks

#   - Multi-chain integration frameworks

#   - Educational tools for PAT understanding
#   - Development utilities and build scripts
